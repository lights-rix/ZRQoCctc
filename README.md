# 项目简介
协同过滤算法+SpringBoot Vue动漫周边商场系统旨在为用户提供一个集推荐、购物于一体的动漫周边商场平台。该系统基于协同过滤算法进行周边商品的智能推荐，提升用户购物体验。

## 核心功能
- 使用协同过滤算法推荐周边商品
- 用户下单功能
- 购物车管理

## 适用场景
- 动漫周边爱好者
- 商场运营者
- 网络零售商

## 技术特点
- 架构：B/S、MVC、Web框架
- 开发环境：IDEA、JDK1.8、Mysql、Maven、Navcat
- 技术栈：Java、SpringBoot、Vue
